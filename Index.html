Pizza Sales Analysis

Project Overview

This project analyzes pizza sales data using SQL queries. It includes insights such as best-selling pizzas, peak order times, and total revenue.

Project Files

  • pizza_sales.sql – SQL script with all analysis queries
  • pizza_sales_database.db – Sample database file (if available)
  • README.md – Project description and instructions

Key Insights (via SQL)

  • Total revenue generated
  • Most popular pizza types
  • Sales by day and month
  • Peak order hours
